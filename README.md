# QiitaApp

概要: QiitaのAPIを使いテーブルビューに記事を表示させたシンプルなアプリです。

使用した技術: Alamofire、AlamofireImage

学んだこと: 

・Alamofireでのリクエスト

・Codableを使ったJSON解析

・JSONSerialization

・クロージャのコールバック
